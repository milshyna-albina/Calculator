# JS Calculator

A clean and functional web calculator with 4 specialized modes for different purposes.

## 🚀 Modes
* **Basic**: Simple arithmetic for everyday tasks
* **Advanced**: Engineering functions (factorial, square root, power)
* **Converter**: Quick unit conversions for various physical quantities
* **Numerical System**: Calculations and conversions between binary, decimal and hexadecimal

## 🛠 Technologies
* **JavaScript**
* **HTML5**
* **CSS3**
